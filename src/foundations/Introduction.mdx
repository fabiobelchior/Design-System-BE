# Design System BE

Welcome to the Design System BE component library.

This library provides a set of reusable React components built on top of a consistent visual foundation derived from our Figma Design System.

## Architecture

The system is organized in two main layers:

### Foundations
The visual base of the design system, including colors, typography, spacing, radius, shadows, and more. These are expressed as CSS custom properties (variables) and TypeScript tokens.

### Components
React components built using the foundation tokens. Each component supports variants, sizes, and visual states as defined in the Figma design.

## Getting Started

Import components directly:

```tsx
import { Button } from '@design-system-be/ui';
```

All design tokens are available as CSS variables:

```css
.my-element {
  color: var(--color-primary-500);
  padding: var(--spacing-lg);
  border-radius: var(--radius-md);
}
```
